  • Quiz

    Create your own quiz app without code.

    A quiz is a form of game or mind sport, attempt to answer questions correctly.



    This app preset can help anyone get set up a quiz app to test knowledge about a certain subject.


    Please follow the instructions below:

  • Step 1

    Open Glowbom Chat:

    Open Glowbom

  • Step 2

    Press + New Project at the top right corner:



    Or press a little mic button and say: "Create an app."


  • Step 3

    Choose Quiz and press Select:



    Or say: "Create a quiz app."

  • Step 4

    Enter project name and press Create project:



    Or say: "Create a quiz app called..." with the name of your project name.

  • Step 5

    To add and modify quiz elements, press a little pencil button:



    Or say: "Open an editor."

  • Step 6

    By default, you have 3 example questions. To add a new question, press "+" button:


    Or say: "Add a quiestion."

    To edit a question, press a question first and then press a little pencil button on top right:

    To remove a question, press a question first and then press a remove button on top left.

    To clear selection, press a selected question second time.

    To close the editor, press a cross button on top center.

  • Step 7

    Fill out the question and answers fields, then mark the correct answers using checkboxes:


    Or say: "The question is...", "The answer is...", "The next answer is...", "The correct answer is..."

  • Step 8

    Press "Play" button on top right to run the project:

    Or say: "Launch the project please."


    Press "Stop" button on top right to stop the project:

    Or say: "Stop the project please."

  • Step 9

    To customize your quiz, open project settings by clicking a small gear button on bottom left:

    You can choose your app color, change copy of the result screen, and make your quiz vocal by enabling voice support:

  • Step 10

    Press "Export" button on bottom right to export *.glowbom project file and the source code:

    Or say: "Export code please."

  • Step 11

    You download data file quiz.glowbom and an archive that has two folders build-web and source-flutter:

    The build-web folder contains a ready to deploy web app.

    The source-flutter folder contains app source code that can be compiled to iOS, Android, and web app.

    Replace build-web/assets/assets/quiz.glowbom file to your quiz.glowbom file:

    Replace source-flutter/app/assets/quiz.glowbom file to your quiz.glowbom file:

  • Step 12

    The build-web folder contains a ready to deploy web app that can be deployed to any hosting.

    We recommend using Firebase, a platform developed by Google:

    Open Firebase

    The source-flutter folder contains app source code that can be compiled to iOS, Android, and web app.

    To compile the source code, you need to install Flutter, a software development kit created by Google:

    Open Flutter
