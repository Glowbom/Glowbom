  • Quiz

    Create your own quiz app without code.

    A quiz is a form of game or mind sport, attempt to answer questions correctly.



    Glowbom Quiz can help you set up a quiz app to test knowledge about a certain subject.


    Please follow the instructions below:

  • Step 1

    Open Glowbom and create an account if you haven't done it already:

    Open Glowbom

  • Step 2

    Press + New Project at the top right corner:



    Or press the little mic button and say: "Create an app."


  • Step 3

    Choose Quiz and press Select:



    Or say: "Create a quiz app."

  • Step 4

    Enter the project name and press Create project:



    Or say: "Create a quiz app called..." with the name of your project name.

  • Step 5

    To add and modify quiz elements, press the little pencil button:



    Or say: "Open an editor."

  • Step 6

    By default, you have 3 example questions. To add a new question, press the "+" button:


    Or say: "Add a quiestion."

    To edit the question, press the question first and then press the little pencil button at the top right:

    To remove the question, press the question first and then press the remove button at the top left.

    To clear selection, press the selected question second time.

    To close the editor, press the cross button at the top center.

  • Step 7

    Fill out the question and answers fields, then mark the correct answers using checkboxes:


    Or say: "The question is...", "The answer is...", "The next answer is...", "The correct answer is..."

  • Step 8

    Press the "Play" button on the top right to run the project:

    Or say: "Launch the project please."


    Press the "Stop" button on the top right to stop the project:

    Or say: "Stop the project please."

  • Step 9

    To customize your quiz, open the project settings by clicking the small gear button on the bottom left:

    You can choose your app color, change copy of the result screen, and make your quiz vocal by enabling voice support:

  • Step 10

    Press the "Export" button on the bottom right to export the quiz.glowbom project file and the source code:

    Or say: "Export code please."

  • Step 11

    You download data file quiz.glowbom and an archive that has two folders build-web and source-flutter:

    The build-web folder contains a ready-to-deploy web app.

    The source-flutter folder contains app source code that can be compiled to iOS, Android, and web app.

    Replace the build-web/assets/assets/quiz.glowbom file with your quiz.glowbom file:

    Replace the source-flutter/app/assets/quiz.glowbom file with your quiz.glowbom file:

  • Step 12

    The build-web folder contains a ready-to-deploy web app that can be deployed to any hosting.

    We recommend using Netlify to publish the web app. It allows you to drag & drop the web app folder directly to the dashboard. The web app deploys automatically, and you will get the app link shortly after that. Optionally, you will be able to connect the web app with the custom domain or subdomain:

    Open Netlify

    To build the app for iOS or Android, you need to install Flutter, a software development kit created by Google. The source-flutter folder contains the Flutter project that can be compiled to iOS, Android, and a web app.

    Install Flutter
