  • Survey

    Create your own survey app without code.

    Collect answers using a Google Sheet.



    This no-code tutorial can help you get set up a simple information gathering app connected to a Google Sheet.


    Please follow the instructions below:

  • Step 1

    Open Glowbom Chat:

    Open Glowbom

  • Step 2

    Press + New Project at the top right corner:



    Or press a little mic button and say: "Create an app."


  • Step 3

    Choose Survey and press Select:



    Or say: "Create a survey app."

  • Step 4

    Enter project name and press Create project:



    Or say: "Create a survey app called..." with the name of your project name.

  • Step 5

    To add and modify survey elements, press a little pencil button:



    Or say: "Open an editor."

  • Step 6

    By default, you have 3 example questions. To add a new question, press "+" button:


    Or say: "Add a quiestion."

    To edit a question, press a question first and then press a little pencil button on top right:

    To remove a question, press a question first and then press a remove button on top left.

    To clear selection, press a selected question second time.

    To close the editor, press a cross button on top center.

  • Step 7

    Fill out the question and answers fields:


    Or say: "The question is...", "The answer is...", "The next answer is...", "The next question is..."

  • Step 8

    Press "Play" button on top right to run the project:

    Or say: "Launch the project please."


    Press "Stop" button on top right to stop the project:

    Or say: "Stop the project please."

  • Step 9

    To customize your survey, open project settings by clicking a small gear button on bottom left:

    You can choose your app color, collect name or phone, and make your survey vocal by enabling voice support:

    To collect answers using a Google Sheet, press "Create Form" button.

  • Step 10

    Make your own copy of "Survey" Google Sheet by going to File->Make a copy:

    Open an editor by going to Tools->Script editor:

  • Step 11

    Clear everything inside the script window:

    Copy a server script:

  • Step 12

    Paste the script code inside the editor and press Save:

    Name the project and press Ok:

  • Step 13

    Publish the script by pressing Publish->Deploy as web app...:

    Choose Anyone, even anonymous in Who has access to the app and press Deploy:

    Press Review Permissions:

    Press Allow:

    Copy Current web app URL:

  • Step 14

    Paste Current web app URL into Answers form link:

    Run the project again and submit the user information:

    The user information should appear in the Google Sheet:

  • Step 15

    Press "Export" button on bottom right to export survey.glowbom project file and the source code:

    Or say: "Export code please."

  • Step 16

    You download data file survey.glowbom and an archive that has two folders build-web and source-flutter:

    The build-web folder contains a ready to deploy web app.

    The source-flutter folder contains app source code that can be compiled to iOS, Android, and web app.

    Replace build-web/assets/assets/survey.glowbom file to your survey.glowbom file:

    Replace source-flutter/app/assets/survey.glowbom file to your survey.glowbom file:

  • Step 17

    The build-web folder contains a ready to deploy web app that can be deployed to any hosting.

    We recommend using Firebase, a platform developed by Google:

    Open Firebase

    The source-flutter folder contains app source code that can be compiled to iOS, Android, and web app.

    To compile the source code, you need to install Flutter, a software development kit created by Google:

    Open Flutter
